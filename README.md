# Projeto Mochila

Este repositório contém o desenvolvimento do projeto **Mochila**, realizado como parte das atividades acadêmicas do curso de **Análise e Desenvolvimento de Sistemas (ADS)** na faculdade **UNIGUAÇU**.

## Objetivo
O objetivo deste projeto é aplicar conceitos de lógica de programação e estrutura de dados.

## Tecnologias Utilizadas
* **Linguagem:** Java
* **Ferramentas:** IntelliJ IDEA, Git e GitHub Desktop.

## Funcionalidades
- [ ] Cadastro de itens.
- [ ] Cálculo de peso/valor.
- [ ] Listagem de objetos na mochila.
